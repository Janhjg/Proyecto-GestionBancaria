# Proyecto-GestionBancaria
## Descripcion:
Este proyecto es un mini sistema bancario en Python que simula operaciones básicas de un banco: creación de usuarios, login, consultas, transacciones y control de saldo. Está diseñado para practicar programación estructurada, uso de funciones, condicionales, validación de datos y manejo de usuarios.

## Funcionalidades Principales

### Gestion de usuario
* Crear nuevo usuario
* Inicio de sesion

### Operaciones bancarias
* Consultar Saldo
* Transferir dinero entre usuarios
