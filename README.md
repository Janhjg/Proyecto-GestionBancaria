# Proyecto-GestionBancaria
## Descripcion:
Este proyecto es un mini sistema bancario en Python que simula operaciones básicas de un banco: creación de usuarios, login, consultas, transacciones y control de saldo. Está diseñado para practicar programación estructurada, uso de funciones, condicionales, validación de datos y manejo de usuarios.


## Funcionalidades Principales

### Menú Gestion de usuario
* Crear nuevo usuario (usuario, PIN)
* Inicio de sesión (Accede a menú de operaciones)
* Salir de aplicación

### Menú Operaciones bancarias
* Consultar saldo
* Ingresar saldo
* Retirar dinero
* Transferir dinero entre cuentas
* Crear cuenta Ahorro/corriente -> Se genera un número de cuenta para cada una
* Salir de sesión (Vuelve a menú de gestión de usuario)
